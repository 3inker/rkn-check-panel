# RKN Check Panel

Удобное меню для пакета [`rkn-block-checker`](https://github.com/MayersScott/rkn-block-checker).

Оригинальный пакет работает только из командной строки. Эта обёртка добавляет простое меню с цифрами.

## 📥 Скачать

**Способ 1 (рекомендуемый)**
1. Перейдите в раздел **Releases** справа
2. Скачайте `rkn.bat`
3. Запустите двойным щелчком

**Способ 2**
Нажмите **Code** → **Download ZIP** → распакуйте → запустите `rkn.bat`

**Важно:** Не скачивайте через кнопку `Raw` — файл не будет работать.

## 🚀 Как пользоваться

Запустите `rkn.bat` и выберите номер нужного действия:

```
============================================================
  rkn-check-panel — Меню управления
============================================================

  [1]  Полная проверка (whitelist + blacklist)
  [2]  Только whitelist
  [3]  Только blacklist
  [4]  Проверить свой URL
  [5]  Полная проверка + JSON
  [6]  Только blacklist + JSON
  [7]  Только whitelist + JSON
  [8]  Быстрая проверка (2 сек, 20 потоков)
  [9]  Медленная проверка (15 сек, 5 потоков)
  [10] Подробный вывод (verbose)
  [11] Отладка (debug)
  [12] Установить / обновить rkn-block-checker
  [13] Проверить установку
  [14] Сохранить отчёт в файл
  [15] Сохранить JSON в файл
  [16] Удалить rkn-block-checker
  [0]  Выход
```

## 📋 Требования

- Windows
- Python 3.10+ (скачать с [python.org](https://python.org/downloads/))
- Интернет

## 🔧 Первый запуск

1. Запустите `rkn.bat`
2. Если Python не установлен — установите
3. Выберите пункт `[12]` — установится основной пакет
4. Пользуйтесь любым пунктом меню
